# QUICK_START.md - 5-Minute Setup Guide

# 🚀 Quick Start - Get Running in 5 Minutes

## Option A: Local Development (Recommended for Testing)

### Step 1: Clone Repository
```bash
git clone https://github.com/yourname/aerial-threat-detection.git
cd aerial-threat-detection
```

### Step 2: Setup Python Environment
```bash
# Create virtual environment
python3.10 -m venv venv

# Activate it
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Upgrade pip
pip install --upgrade pip
```

### Step 3: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 4: Download Models
```bash
# Models auto-download on first use, or download manually:
python -c \"from ultralytics import YOLO; YOLO('yolov8x.pt')\"
```

### Step 5: Run API Server
```bash
python src/api/main_api.py
```

**✅ Done!** API is now running at `http://localhost:8000`

### Test It
```bash
# Open browser
open http://localhost:8000/docs

# Or test with curl
curl http://localhost:8000/health
```

## Option B: Docker (Recommended for Production)

### Step 1-2: Clone & Navigate
```bash
git clone https://github.com/yourname/aerial-threat-detection.git

```

### Step 3: Start Full Stack

**✅ All services running!**

- **Frontend**: http://localhost:3000
- **Database**: localhost:5432

### Monitor
```bash

```

## Python Usage

### Quick Detection Script
```python
import requests

# Detect threats
with open('image.jpg', 'rb') as f:
    response = requests.post(
        'http://localhost:8000/detect/frame',
        files={'file': f}
    )

results = response.json()
print(f\"Found {results['threats_detected']} threats\")

for threat in results['threats']:
    print(f\"  {threat['class']}: {threat['threat_level']}\")
```

### Real-time Monitoring
```python
import asyncio
import websockets

async def monitor():
    async with websockets.connect('ws://localhost:8000/ws') as ws:
        while True:
            msg = await ws.recv()
            print(msg)

asyncio.run(monitor())
```

## Troubleshooting

### 1. Port Already in Use
```bash
# Change port in code or use:
export API_PORT=8001
python src/api/main_api.py
```

### 2. CUDA Not Found
```bash
# Check GPU
python -c \"import torch; print(torch.cuda.is_available())\"

# Run on CPU instead
export CUDA_VISIBLE_DEVICES=
python src/api/main_api.py
```

### 3. Out of Memory
```bash
# Use smaller model
# Edit threat_detection.py: change 'yolov8x.pt' to 'yolov8m.pt'
```

### 4. Models Not Downloading
```bash
# Manual download
wget https://github.com/ultralytics/assets/releases/download/v0.0.0/yolov8x.pt -O models/yolov8x.pt
```
